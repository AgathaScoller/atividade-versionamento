# Atividade de Versionamento - Conclusão

Este repositório contém os arquivos e etapas realizadas por mim durante a atividade de versionamento com Git e GitHub.

## Etapas que realizei

1. Criei um repositório local e inicializei o Git.
2. Criei o arquivo **index.txt** com a versão inicial contendo `<H1> TÍTULO1 </H1>`.
3. Publiquei essa versão na branch **main**.
4. Criei a branch **feature1** e publiquei a mesma versão lá.
5. Altei a linha do H1 na main para **VERSIONAMENTO** e enviei o commit.
6. Altei a linha do H1 na feature1 para **GIT** e enviei o commit.
7. Realizei o merge da feature1 na main.
8. Resolvi manualmente o conflito gerado entre as duas versões.
9. Finalizei o merge e publiquei o resultado.

## Arquivos incluídos neste repositório

- `index_inicial.txt` — Versão inicial criada.
- `index_main.txt` — Versão alterada na branch main.
- `index_feature1.txt` — Versão alterada na branch feature1.
- `index_resolvido.txt` — Versão final após resolução do conflito.

Toda a atividade foi realizada seguindo o fluxo de versionamento solicitado.
